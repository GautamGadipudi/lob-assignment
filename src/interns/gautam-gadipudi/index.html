Background

Prior to starting my masters degree, I did my bachelors degree also in Computer Science at Manipal Institute of Technology back in India. In my final semester there, when I was offered an internship at DeltaX, I was finally able to see the big picture of everything I learned in my college. DeltaX is an AdTech startup in Bengaluru with their focus on their flagship product - DeltaX platform. During my 6 months of internship there, I developed a service to fetch the Facebook lead forms from the Facebook marketing API, store them in AWS S3 as log files and finally export them to appropriate Google sheets using Google sheets API for C#. This added a feature for the clients to be able to view their data directly in Google sheets.

After I graduated, I was offered a full time role as a Full Stack Developer at DeltaX, and I worked there for close to a year. Since it was a start-up and we worked in an ownership driven kind of environment, I got an exposure to the complete tech stack at DeltaX. I was part of the tracking team, where our main responsibility was to provide timely and demographic ad performance data using metrics like clicks, views, cost per click etc. for the different ads and campaigns of different clients. My learnings from there were developing web applications using .NET and Vue.js, and developing microservices using C#, testing them using Behavior Driven Development and deploying them using Jenkins and Azure. One breakthrough achievement of mine there was to reduce the load time of web pages showing computed grid data by a very huge margin - earlier it used to take about a minute to load the grid data for big clients with large audience like ClearTrip and Dominos pizza because the SQL query to compute the grid data was the bottleneck, but with the implementation of a redis cache to store timely computed data in it, I was able to reduce the load time to a couple of seconds.
